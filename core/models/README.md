# Core Models

## Responsabilidade

Este pacote reúne os modelos-base compartilhados pelo sistema.

## Escopo desta missão

- Criar contratos iniciais para entidades centrais
- Preparar um pacote tipado e validável
- Servir de fundação para futuras missões

## O que este módulo não faz

- Não implementa lógica de negócio
- Não acopla modelos a engines
- Não cria persistência
- Não define regras de workflow

## Modelos-base previstos

- `Project`
- `Video`
- `Script`
- `Asset`
- `PipelineJob`
- `EngineResult`
- `Metadata`

## Observação arquitetural

Os modelos aqui definidos são placeholders estruturais e podem ser ampliados em
missões futuras sem quebrar a separação de responsabilidades.
