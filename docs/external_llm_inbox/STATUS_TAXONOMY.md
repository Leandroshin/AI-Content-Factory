# Taxonomia de Status

Use exatamente um destes rotulos em documentos externos:

- `PROPOSTA - NAO IMPLEMENTADA`
- `PESQUISA - NAO VALIDADA`
- `PROTOTIPO ISOLADO - NAO INTEGRADO`
- `CANDIDATO A EVIDENCIA - NAO AUDITADO`
- `AUDITORIA PARCIAL - FALTA CORROBORACAO`
- `EXPERIMENTO MOCK - SEM EFEITO OPERACIONAL`
- `INTEGRADO PELO CODEX - TESTADO`
- `REAL CONTROLADO - OPT-IN E APROVACAO`
- `REJEITADO - NAO USAR`

Proibido usar `PRONTO`, `IMPLEMENTADO`, `FUNCIONANDO` ou `OFICIAL` para uma proposta/prototipo.
