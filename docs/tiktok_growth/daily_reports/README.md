# Daily gaming radar reports

The scheduled Gaming News Desk writes a dated report here only when a new,
relevant and verifiable opportunity exists.

An empty-news run updates deduplication state but does not create a content
brief or owner notification.

Each report must contain:

- original source and publication date;
- primary-source verification status;
- novelty versus prior reports;
- why the topic matters now;
- hook and visual plan;
- copyright, rumor and policy risks;
- TikTok and YouTube Shorts recommendation;
- explicit owner approval state.
